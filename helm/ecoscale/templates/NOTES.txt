EcoScale has been deployed to {{ .Release.Namespace }}.

The World's First Carbon-Aware Kubernetes Controller is now running.

Access the service:
  kubectl port-forward -n {{ .Release.Namespace }} svc/{{ include "ecoscale.fullname" . }} 8080:8080

Then visit:
  - http://localhost:8080/          - API info
  - http://localhost:8080/health    - Health check
  - http://localhost:8080/metrics   - Prometheus metrics (CO2_Saved_Total, etc.)
  - http://localhost:8080/recommendations - Live optimization recommendations

To mark workloads as carbon-flexible, add the label to your pods:
  ecoscale/flexible=true
